AI4SoilHealth App

AI4SoilHealth App is a client Vue PWA application.
Its design is modular, and is mostly driven with parameters stored in tables of Meta schema in the AI4SoilHealth database. It communicates with the database usin Rest API calls of controllers in AI4SoilHealth backend.

This site contains description of all modules and functions of the client app.
